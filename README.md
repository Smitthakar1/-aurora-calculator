- ✨ Ambient animated glow border

- 💡 Live expression display
- 📜 Rolling calculation history
- ⌨️ Full keyboard support

---

## 🚀 Getting Started

### Requirements
- Python 3.x  
- `tkinter` — **already included** with Python on Windows & macOS.  
  On Linux: `sudo apt install python3-tk`

### Run it
```bash
git clone https://github.com/YOUR_USERNAME/aurora-calculator
cd aurora-calculator
python calculator.py
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Digits |
| `+ - * /` | Operators |
| `Enter` | Equals |
| `Backspace` / `Esc` | Clear |
| `.` | Decimal |
| `%` | Percent |
